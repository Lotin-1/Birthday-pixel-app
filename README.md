# Birthday-pixel-app
A cute pixel-style interactive birthday web app built with Vue.js.  Users can choose between different pixel art icons (cake, cupcake, gift, party) and receive a personalized birthday message.  The app includes aesthetic UI design, soft animations, and floating bubble effects for a cozy and emotional experience.

# 🎂 Pixel Birthday Surprise

A cute interactive birthday web app made with Vue.js 💖

## ✨ Features
- Pixel art aesthetic
- Interactive selection (cake, cupcake, gift, party)
- Personalized birthday messages
- Floating bubble animations

## 🚀 Technologies
- HTML
- CSS
- JavaScript
- Vue.js

## 💌 Purpose
This project was created as a creative and emotional digital birthday experience.
